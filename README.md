## Points à revoir / améliorer

### Feedback tactile (mobile)
- Ajouter un vrai effet de “push” sur les miniatures (`scale`, `shadow`, `opacity`)
- Désactiver le tap highlight par défaut sur mobile (Android)

### Titres
- Rôle principal : **rythmer la page**, pas dominer le contenu
- Tester un élément graphique discret (filet fin, marqueur léger)
- Ne pas ajouter de titre pour la section de transition

### Accessibilité (à traiter dans un second temps)
- Rendre les œuvres navigables au clavier
- Utiliser `:focus-visible`
- Vérifier la hiérarchie et la lisibilité sans surcharger l’interface
